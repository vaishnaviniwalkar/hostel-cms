# 🏫 Hostel Complaint Management System

A simple and efficient web application for students to submit hostel complaints and for wardens to manage and resolve them. Built using **HTML, CSS, JavaScript, Firebase Authentication, and Firestore**.

🔗 **Live Demo:**  
https://hostel-cms.web.app 

---

## 🚀 Features
- Student & Warden Login/Signup  
- Raise new complaints  
- Track complaint status in real-time  
- Warden panel to view and update complaints  
- Firebase Auth + Firestore backend  
- Clean and responsive UI  

---

## 🛠 Tech Stack
- HTML  
- CSS  
- JavaScript  
- Firebase Authentication  
- Firebase Firestore  
- Firebase Hosting  

---

## ▶️ How to Run Locally
1. Clone the repository  
2. Open the project in VS Code  
3. Right-click `index.html` → **Open with Live Server**  
4. Firebase config is inside `js/firebase.js`  

---

## 🚀 Deploy to Firebase
- firebase init hosting
- firebase deploy
